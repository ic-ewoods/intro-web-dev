Structure of HTML Elements

Most elements only belong in certain places within the structure of an HTML document. This guide tries to help make that clear.

Sections

Sections are for grouping text containers. Don't put text directly in them.

  • body
  • section
  • article
  • nav
  • header
  • footer

Text containers

If you want to add text, make sure it's in one of these.

  • h1..h6
  • p
  • li (only inside an ol or ul)
  • dt, dd (only inside a dl)
  • blockquote
  • div

Special text containers

These have special constraints

  • li, dd, blockquote, div - can contain other text containers
  • hr - empty, so it has no text content
  • ol, ul - direct children should only be li elements
  • dl - direct children should only be dt and dd elements

Text elements

Inside a text container, you can also add this markup.

  • b / strong
  • i / em
  • sup, sub
  • abbr
  • br
  • a
  • span
  • img

Embedded content

There are several kinds of embedded content can be used inside text containers and text elements, but they can also be outside of text containers. Frequently these kinds of elements will still be within a general purpose div container.

  • img
  • iframe
  • embed
