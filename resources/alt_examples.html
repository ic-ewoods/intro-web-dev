img alt tag examples

An img tag must have an alt attribute that is an appropriate replacement for the image.

The examples below demonstrate application of the following helpful ideas for writing alt tags:

"In general, alternative text can be written by considering what one would have written had one not been able to include the image."
"One way to think of alternative text is to think about how you would read the page containing the image to someone over the phone, without mentioning that there is an image present. Whatever you say instead of the image is typically a good start for writing the alternative text."

Both of these recommendations are from the HTML Living Standard guidelines for use of alt tags
