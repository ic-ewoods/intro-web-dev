WebStorm Project Password Instructions (Mac only)

These instructions can be used only on a Mac or on lab computer (by using Cygwin).

Change Your Password

You have an account on the comp105ithaca.com server. But we need you to have a more secure password.

  1. Inside WebStorm, go to the menu Tools Open Terminal...
  2. Type the following command at the prompt and hit return: 
    ssh IC_USERNAME@comp105ithaca.com
    but replace "IC_USERNAME" with your IC login/username. It will ask you if you trust the server (type yes and hit return).
  3. It will prompt for your password. Type your first name with standard capitalization and hit return. (This is your first name as it appears in Homer.)
    No characters appear while you are typing your password -- this is normal.
  4. When you have successfully logged in you will get a response that end with a line similar to this:
    comp105:~$
  5. Change you password by typing 'passwd', hitting return, and following the instructions
    comp105:~$ passwd
    Like logging in, you don't see the characters you are typing while you enter your password.
    Please use a strong password (include some things like capitals, numbers, and special characters).
  6. Once your new password is accepted you should see that it says the password updated successfully and you can close the terminal panel (using the red 'X' on the left).
    woods@comp105:~$ passwd
Changing password for woods.6
(current) UNIX password:
Enter new UNIX password:
Retype new UNIX password:
passwd: password updated successfully
woods@comp105:~$
